== Description ==
SKPlanet에서 제공하는 실시간 댓글 플러그인을 여러분의 wordpress에 설치하여 사용할수 있습니다.

== Installation ==
SKPlanet 개발자센터(https://developers.skplanetx.com)에서 계정을 생성한후. Affogato서비스 APP_ID를 발급받아 wordpress plugin을 설치후 환경설정>Affogation 설정에서 APP_ID를 설정하시기 바립니다.
